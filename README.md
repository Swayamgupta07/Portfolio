## 🎨 Premium Animation Stack & Key Features

This portfolio is packed with interactive design systems, fluid micro-interactions, and premium Apple-style aesthetics.

### 1. Buttery-Smooth Scroll System
- **Lenis Smooth Scroll**: Integrates Virtual Smooth Scrolling to eliminate standard browser page jumps and deliver a consistent fluid scroll feel.
- **GSAP Ticker Sync**: Lenis scroll physics are linked directly with the **GSAP Animation Ticker loop** to ensure ScrollTriggers, timelines, and visual updates trigger with zero frame lag.
- **IntersectionObserver ScrollSpy**: Real-time viewport observation dynamically highlights the active navigation menu item (`#home`, `#about`, `#projects`, etc.) with sliding underlines.

### 2. High-Performance Cursors & Overlays
- **GSAP quickTo Cursor**: A customized mouse-following dot and outer circle. Uses high-performance `gsap.quickTo` functions to update positioning vectors instantly without micro-stutters.
- **Magnetic Link Hover**: Cursors automatically scale up and apply radial magnetic pulls when hovering over links, buttons, or custom pointer indicators.
- **Dark Mode Mouse Spotlight**: Paints a radial halo overlay of light following the cursor in dark mode to spotlight active screen sections.
- **Canvas Constellation Particles**: An interactive, full-screen canvas background that floats particles in random vectors and pushes them away when the mouse gets near.

### 3. Apple-Style entrance & Scroll reveals
- **System Loading Portal**: Counts from 0% to 100% with progress bar indicators, splitting and sliding the viewport panel open vertically upon completion.
- **3D Tilt Hover Effects**: Profile cards use **Framer Motion spring physics** to tilt on three dimensions depending on cursor placement.
- **Progressive Timeline Draw**: The timeline line automatically fills and draws itself downward as the user scrolls, triggering staggered entrance cards in sequence.
- **Auto-Drawing Progress Bars**: Skills proficiency levels dynamically load from `0%` to their target parameters as the section enters the viewport.
- **Animated Numerical Stats**: Project counters and experience indicators count up dynamically from zero.

## 🚀 Installation & Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Swayamgupta07/professional_portfolio_website.git
   cd professional_portfolio_website
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the local dev server**:
   ```bash
   npm run dev
   ```

4. **Compile production build**:
   ```bash
   npm run build
   ```
